This plugin integrate Google WebFonts v2 to your website. Support nearly all of Google Webfonts.

== Description ==

This plugin integrate Google WebFonts v2 to your website.
You can Select 2 different Font for your two different area. You can apply with this fonts using CSS codes. Look at admin area screenshot.
For More Information [WPAdami, Serkan Algur](http://wpadami.com/cms-sistemleri/wordpress/google-webfonts-integrate.html)

Preview Mode added. You can decide which font is more beautiful :)

With version 0.5: Subset selector added. Plugin support more subsets. All fonts added except Khmer subset and Condensed fonts. Workin' on it.

With version 0.6: You can add your own font from Google Web Fonst. You can paste all link which give you from google. Also you can use styling with Font 1 CSS area.

With version 0.7: Nothing changed yet. Only compatibility check.

With version 1.0 : You have 500+ fonts now!

== Installation ==

1. Upload `google-webfonts-integrate` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
